## Retrospective Schedule

Retrospectives run after each milestone or every 2-3 dispatch cycles (whichever comes first).

**Format:**
1. What went well?
2. What didn't go well?
3. What tensions were raised?
4. What process improvements to try next cycle?

Records saved in `00_Organization_Logbook/Reviews/`.

## Agent Evaluation Protocol

For each AI agent, Alex reviews quarterly:
1. **Output quality** — against the role's key metrics
2. **Boundary compliance** — any escalation failures or policy violations?
3. **Tension sensing** — did the agent raise useful tensions?
4. **Driver alignment** — did the agent's work serve the primary driver?
