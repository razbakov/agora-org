# Policy 003: Open Source

**Date:** 2026-04-07
**Status:** Active

## Purpose
Ensure the Ikigai Team framework remains open and accessible as public infrastructure.

## Intended Outcome
All core framework code and governance research are freely available. No one is locked into proprietary governance tools.

## Policy Details
1. **Core framework is MIT licensed.** The setup flow, agent definitions, coaching flow, and delegation protocol are open-source.
2. **Research is shared openly.** Research summaries, protocol specs, and governance model analyses are public.
3. **Governance docs are public.** This organization's logbook is open for transparency and as a reference implementation.
4. **Premium features are additive.** Any commercial features (hosting, advanced tooling) build on top of the open core — they don't gate core functionality.
5. **Attribution.** When building on others' work (Democracy Earth, S3), cite sources.

## Responsibilities

| Who | What |
|-----|------|
| Builder | Ensure code stays MIT licensed, no proprietary deps in core |
| Researcher | Publish research openly |
| Alex | Decide boundary between open core and premium features |

## Metrics and Monitoring

| Metric | How Monitored | Frequency |
|--------|--------------|-----------|
| License compliance | Code review | Per PR |
| Research accessibility | All summaries in public repo | Monthly |
